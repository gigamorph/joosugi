# Joosugi

Joosugi aims to be a library that acts as a [IIIF](http://iiif.io)-compliant interface to annotation data store(s).

## Install
```
npm install joosugi
```

## Use

```javascript
var model = new joosugi.AnnotationExplorer({
  dataSource: dataSource
});
```
